# Project-2-Random-Dice-First-Coding-
/This project is written in Python/ In this project we have two dice that are thrown randomly. The way it works is that when both dice are thrown, each one displays a number. If the numbers are equal, it displays the word won. If they are not equal, it displays the word lost. This was my first project in college.


# Random Dice Game

A simple Python project that simulates rolling two dice.

## Features
- Generates two random dice values
- Displays "Won" if both dice are equal
- Displays "Lost" if the dice values are different

## Technologies
- Python
- random module
